# 雙策略選股系統設計

## 策略對比

| 項目 | 短線操作 | 投資佈局 |
|------|---------|---------|
| **時間** | 1-2週 | 1-6月 |
| **重點** | 技術面 70% + 基本面 30% | 基本面 70% + 技術面 30% |
| **停損** | -5~8% | -15~20% |
| **目標** | +10~20% | DCF 內在價值 |
| **勝率** | 60-70% | 80%+ |
| **適合** | 積極交易者 | 價值投資者 |

## 短線策略實作

### 評分權重
```python
SWING_WEIGHTS = {
    'trend': 0.35,      # 趨勢（MA 排列）
    'momentum': 0.25,   # 動能（RSI/MACD/KD）
    'volume': 0.15,     # 成交量
    'breakout': 0.15,   # 突破訊號
    'fundamental': 0.10 # 基本面加分
}
```

### 篩選流程
1. **初篩**：MA5 > MA10 > MA20
2. **動能確認**：RSI 50-70, MACD 黃金交叉
3. **量價配合**：成交量放大
4. **排序**：綜合評分 Top 10
5. **輸出**：含進場價、停損、目標價

## 投資策略實作

### 評分權重
```python
INVEST_WEIGHTS = {
    'dcf_margin': 0.40,     # DCF 安全邊際
    'profitability': 0.15,  # 獲利能力（ROE/EPS成長）
    'safety': 0.15,         # 財務安全（負債比）
    'trend': 0.20,          # 技術面確認底部
    'valuation': 0.10       # 其他估值指標
}
```

### 篩選流程
1. **初篩**：DCF 安全邊際 > 30%
2. **財務健檢**：ROE > 10%, 負債比 < 60%
3. **底部確認**：RSI > 30, 不在持續下跌
4. **排序**：安全邊際由高到低
5. **輸出**：含內在價值、安全邊際、預期報酬

## CLI 使用方式

```bash
# 短線選股
python morning_run.py --strategy swing

# 投資佈局
python morning_run.py --strategy invest

# 雙策略並行（預設）
python morning_run.py --strategy both
```
