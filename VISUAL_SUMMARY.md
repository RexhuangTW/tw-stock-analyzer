## 🎨 使用情境範例

### 情境 1: 找出超跌反彈機會
```python
from src.screener.strategies import screen_rsi_oversold

# 在科技股中找 RSI < 25 的標的
tech_stocks = ['2330', '2454', '2303', '3711', '2408']
oversold = screen_rsi_oversold(tech_stocks, rsi_threshold=25)

if not oversold.empty:
    print("超跌標的:", oversold['stock_id'].tolist())
```

### 情境 2: 監控持股突破
```python
from src.monitor.watcher import StockWatcher

watcher = StockWatcher(check_interval=300)
watcher.add_stock('2330', {'price_above': 2100})  # 台積電突破 2100
watcher.add_stock('2454', {'rsi_below': 30})       # 聯發科 RSI 超賣
watcher.start_monitoring()  # 每 5 分鐘檢查
```

### 情境 3: 批次掃描
```python
from src.screener.strategies import screen_golden_cross

# 掃描 50 檔股票找黃金交叉
stock_pool = ['2330', '2317', '2454', ...]  # 50 檔
result = screen_golden_cross(stock_pool)

print(f"找到 {len(result)} 檔黃金交叉!")
print(result[['stock_id', 'date', 'close', 'ma5', 'ma20']])
```
